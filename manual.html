✅ Screenshot Paste — Step by Step

1. Open CIQ factsheet → Screenshot the Key Financial Metrics table → Ctrl+C
2. In the sidebar, click "📋 Paste Table Screenshot"
3. Press Ctrl+V anywhere on the page
4. Status shows: "✅ Screenshot 1 merged! Rev: $713,163M..."
5. Repeat for Liquidity Metrics table (data merges automatically)

ℹ️ What each screenshot captures

  • Key Financial Metrics table → Revenue, EBITDA, Net Profit, Cash, Debt, Equity, FCF, all ratios (CCC, Net Debt/EBITDA, EBITDA/Interest, Current Ratio), quarterly data
  • Liquidity Metrics table → Facility headroom (annual + quarterly if shown)
  • You can paste multiple screenshots — data merges automatically

⚠️If screenshot shows $0M

CIQ Excel files need to be opened in Excel first (to load live data), then saved, then screenshotted. If you paste directly without opening in Excel, the values will be empty.

4
Generate the CM
⚡

Click "⚡ Generate" in the toolbar

The AI fetches financials (if no CIQ data) and writes the full analysis. Takes ~40-60 seconds. Status shows: "⏳ Fetching financials... → ⏳ Writing analysis..."

📋

If a saved draft exists — choose your path

You'll see: "📂 Continue draft" or "⚡ Fresh search"
→ Click "⚡ Fresh search" to generate with latest data (recommended)
→ Click "📂 Continue draft" to load your last saved work

✅

Generation complete

All sections fill automatically: financial table, key highlights, executive summary, recommendation, risks, source of repayment, action plan. Today's date fills in the header. Rating color is applied.

5
Review & Edit

After generation, every field is editable — click any text, number, or dropdown to change it.

🔢

Financial numbers — Section 5 Key Financial Metrics

Click any number to edit. Numbers show with thousands separators (648,125). The YoY % and ratios auto-recalculate. Column headers (Jan-24, Jan-25...) can also be clicked to rename.

📝

Text boxes — Executive Summary, Recommendation, Key Highlights

Click inside any box to edit. Recommendation paragraph labels ("Business Profile", "Outlook" etc.) can also be clicked and renamed.

🎯

Rating — Current / Proposed

Click the dropdown to change (A/B/C/D). The color updates automatically: A=Green | B=Amber | C=Orange | D=Red

⚠️

Risks section — Section 9

The first row is always "Source of Repayment". Rows 2-4 are risks. Each row has a Risk Level dropdown (High/Medium/Low), editable description, and editable mitigant. Click "+ Add Risk Row" to add more.

7
Save & Load Drafts
💾

Save Draft

Click "💾 Save Draft" in the toolbar. If a draft already exists, you'll be asked to override or save as a new version. Drafts are saved to Notion automatically.

📂

Load Saved Draft

1. Type buyer name in the search box
2. Click "⚡ Generate"
3. If a draft exists → you'll see "📂 Continue draft" and "⚡ Fresh search"
4. Click "📂 Continue draft" to restore your saved work

⚠️Important: Always use Fresh Search for updated financials

If you want the latest AI-generated financial data, always click "⚡ Fresh search". Loading an old draft will show the financial figures from when you last saved.

8
Export to PDF
🖨

Click "🖨 Export PDF" in the toolbar

The browser's print dialog opens. Select "Save as PDF" as the destination. All input values are converted to visible text automatically before printing.

✅

Export also auto-saves your draft

Every export automatically saves the current state to Notion. No need to manually save before exporting.

💡Best PDF results

• Make sure all financial data is filled in before exporting
• Review all text sections first
• In the print dialog: set margins to "Default" or "Minimum"
• Page size: A4

Troubleshooting
ProblemSolution
Page looks outdated / changes not showing Press Cmd+Shift+R (Mac) or Ctrl+Shift+R (Windows) for a hard refresh
Financial table shows 0 after loading draft Click "⚡ Fresh search" instead of "Continue draft" to regenerate with latest data
Pasted screenshot shows $0M in status Open the CIQ file in Excel first (so CIQ loads the data), then take the screenshot
Sidebar not visible Click the ☰ button in the top-right corner
Recommendation boxes empty after loading draft Old drafts (before June 24) don't have rec content. Generate fresh with "⚡ Fresh search"
Liquidity metrics period shows FY2022/2023/2024 Click "⚡ Fresh search" to regenerate. Periods sync to the financial table after generate
PDF missing financial numbers Generate fresh before exporting. Old drafts may have empty financial inputs
Rating color not showing Click the rating dropdown and select a value (A/B/C/D) — color applies on selection
