✅ How to use Screenshot Paste — step by step

  1. Open CIQ factsheet in Excel → find the Key Financial Metrics table
  2. Take a screenshot of the table → press Ctrl+C (copy to clipboard)
  3. Click the purple "📋 Paste Table Screenshot" button in the sidebar (see screenshot above)
  4. Press Ctrl+V anywhere on the page → wait ~10 seconds for AI to process
  5. Status bar shows: "✅ Screenshot 1 merged! Rev: $713,163M..."
  6. Repeat for Liquidity Metrics table — data merges automatically

💡 You can paste multiple screenshots

Paste the main financial table first, then paste the liquidity metrics table separately. Each paste adds to the data without losing previous entries. The status bar shows "Screenshot 2 merged..." etc.

⚠️ Screenshot shows $0M?

CIQ Excel files only show data when opened in Excel with CIQ add-in active. Take your screenshot directly from the open Excel file — not from a saved file sent via chat.

3
Generate the Credit Memo
1

Click "⚡ Generate" in the top toolbar

Takes ~40-60 seconds. Status: "⏳ Fetching financials... → ⏳ Writing analysis..."

2

If a saved draft exists — choose your path

"📂 Continue draft" = loads your last saved work
"⚡ Fresh search" = generates fresh with latest data (recommended for updated financials)

3

Generation complete — all sections are filled

Financial table, key highlights, executive summary, recommendation, risks, source of repayment, action plan all written. Review and edit as needed.

4
Save, Load & Export
💾

Save Draft

Click "💾 Save Draft" → saved to Notion. Exporting PDF also auto-saves.

📂

Load Saved Draft

Type buyer name → Generate → "📂 Continue draft" to load. Or "⚡ Fresh search" for new generation.

🖨

Export PDF

Click "🖨 Export PDF" → browser print dialog → Save as PDF. All numbers are visible. Also auto-saves draft.
